**Trouve le digicode !**

Un digicode bloque la porte d'entrée.

Tu as pourtant mis la main sur la liste des digicodes du quartier.

Ils sont dans un carnet mais les adresses ne sont pas dessus.

Tu sais pourtant que le code recherché est un nombre pair

divisible par 7, contenant 6 chiffres, que le nombre suivant contient le chiffre 4

et qu'aucun code parmi les trois précédents ne contient le chiffre 3.


En entrée : une liste de code.

Le résultat attendu : le code recherché.

200 points pour la bonne réponse !
